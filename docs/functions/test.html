APPX

Windows Package Management Functions

FUNCTION

appx or appx.path

Purpose: Returns the path of Package.

Syntax

appx(packageName)
appx.path(packageName)

Parameters

packageName
can be passed in full name or part of name.

Example

Code
appx.path("WindowsTerminal")
// result:
// C:\Program Files\WindowsApps\Microsoft.WindowsTerminal_1.11.3471.0_x64__8wekyb3d8bbwe
FUNCTION

appx.name

Purpose: Returns the display name of the Package.

Syntax

appx.name(packageName)

Parameters

packageName
can be passed in full name or part of name.

Example

Code
appx.name("WindowsTerminal")
// result:
// Windows Terminal
